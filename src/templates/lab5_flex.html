Верстка web-страницы – это процесс ее формирования по готовому макету. Макет создается с использованием графических редакторов, часто в настоящее время для этих целей используется приложение Figma, представляющее собой кроссплатформенный онлайн-сервис для дизайнеров интерфейсов и веб-разработчиков.

Грамотная верстка должна учитывать различное разрешение мониторов пользователей, многообразие браузеров, обеспечивать выделение логических блоков, создавать компактный и красивый код. В настоящее время используется блочная верстка. В последние годы широкую популярность приобрели технологии Flexbox и CSS Grid. С их помощью можно достаточно просто обеспечить резиновый, адаптивный, а также отзывчивый дизайн. Данная лабораторная работа посвящена изучению технологии Flexbox, а также использованию медиазапросов, позволяющих перестраивать макет сайта с учетом размера экрана устройства, на котором отображается web-страница. Результатом их совместного использования является отзывчивый дизайн.

По высоте можно выравнивать и отдельно взятый элемент. Для этого предназначено свойство align-self, у которого возможные значения такие же, как и у свойства align-items. Можно сказать, что свойство align-self для конкретного элемента переопределяет значение выравнивания, заданного для flex-контейнера свойством align -items. Следует заметить, что если для элемента установлено свойство margin со значением auto, то свойство align-self работать не будет. Для того, чтобы применить это свойство к конкретному flex-элементу, расположенному во flex-контейнере, можно, определяя CSS стиль данного элемента, использовать следующую форму записи:
